# Chapter 4 · Lesson 2 — dependencies for the query service.
#
# The query service retrieves context, reranks it, and calls the LLM. It does
# NOT parse PDFs, so it carries none of the heavy Docling / vision stack.
#
# It DOES rerank retrieved chunks with a cross-encoder
# (cross-encoder/ms-marco-MiniLM-L-6-v2), which requires sentence-transformers
# (and, transitively, torch). That model is pre-cached at build time in
# Dockerfile_Query so the first query doesn't download it at runtime.
